# Brouillons du blog

Les articles que la routine `redacteur-blog-tjd` a produits mais que le vérificateur du fond a marqués `DOUTE` atterrissent ici.

Ils sont poussés sur `origin` mais **jamais en prod**, donc invisibles du public (le dossier est aussi bloqué dans `robots.txt`).

Pour publier un brouillon, dis à Claude « publie le brouillon sur [sujet] ». Il corrige le point signalé, déplace l'article à la racine, met à jour le sitemap et la page blog, et pousse en prod.
